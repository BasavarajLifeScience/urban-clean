# Society Booking Admin Dashboard

Admin dashboard for managing the Society Booking Application.

## Getting Started

### Prerequisites
- Node.js 20+ LTS
- Backend API running on port 5001

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the admin dashboard.

### Login Credentials

```

Password: Admin@123
```
